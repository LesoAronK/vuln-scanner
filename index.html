⚠ This tool performs passive, browser-based checks only. No actual port scanning or intrusive probing is performed. Use only on targets you own or have permission to test.

What Gets Checked

🛡

Security Headers

Validates presence of CSP, HSTS, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, and Permissions-Policy.

🔌

Common Ports

Simulates checks for commonly exposed services: SSH (22), FTP (21), Telnet (23), HTTP (80), HTTPS (443), MySQL (3306), and more.

📦

Software Versions

Detects version disclosures in Server headers, X-Powered-By, and common CMS indicators that reveal outdated stacks.

🔒

SSL / HTTPS

Checks if the site enforces HTTPS, has a valid SSL certificate, and doesn't serve mixed content over insecure HTTP.
