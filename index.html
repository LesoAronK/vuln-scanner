Scanner
This tool performs passive, simulated checks only. No real network traffic is sent to the target host. For educational use — only test systems you own or have explicit permission to assess.

What gets checked

Security headers

Validates CSP, HSTS, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, and Permissions-Policy.

Port exposure

Checks for publicly exposed FTP, Telnet, SSH, MySQL, Redis, MongoDB, and other common services.

Software versions

Detects version disclosures in Server headers, X-Powered-By, CMS generators, and more.

SSL / TLS

Checks HTTPS enforcement, weak cipher suites, legacy TLS versions, and certificate expiry.

Misconfigurations

Scans for debug mode, directory listing, exposed .git, wildcard CORS, and robots.txt leaks.

Full reports

Every finding includes a description, severity, remediation steps, and relevant CVE references.
